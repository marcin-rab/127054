Internet applications final project



Objective

The aim of this project is to present various techniques used for web development. It was designed during the Internet applications course, held by Mr. Rafał Różycki at Poznań University of Technology. The course covered the basics of popular web technologies such as HTML5, JavaScript, PHP, MySQL, canvas elements, SVG, MathJAX, webstorage, geolocation, webworkers and essentials of mobile apps.
